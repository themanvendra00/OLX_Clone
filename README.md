# MAD - Make a Deal

A fully responsive **OLX-inspired marketplace web app** where users can browse, search, and view product listings. Built using **HTML, CSS, and JavaScript**, this project focuses on replicating the core UI and functionality of a real-world classifieds platform.

---

## Live Demo  
🔗 https://gleeful-chaja-f20a1c.netlify.app/

---

## GitHub Repository  
🔗 https://github.com/themanvendra00/OLX_Clone

---

## Features

- Browse product listings  
- Search and filter items  
- Detailed product view pages  
- Fully responsive design  
- Smooth and interactive UI  

---

## Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (ES6)** – Logic & interactivity  

---

## Purpose of the Project

This project was built to:

- Practice building a **real-world marketplace UI**
- Improve **DOM manipulation and event handling**
- Strengthen **responsive design skills**
- Understand how listing-based platforms work  

---

## Learnings

- Managing dynamic UI updates with JavaScript  
- Implementing search and filtering logic  
- Structuring scalable frontend code  
- Creating reusable UI components  

---

## ⚙️ How to Run Locally

```bash
# Clone the repository
git clone https://github.com/themanvendra00/OLX_Clone.git

# Navigate into the project
cd OLX_Clone

# Open index.html in browser
```
